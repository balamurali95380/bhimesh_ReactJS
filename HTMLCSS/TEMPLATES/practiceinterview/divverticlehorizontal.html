here are several methods to center a div both vertically and horizontally using CSS. Below are some common and effective ways to achieve this: 1. Using Flexbox Flexbox is a powerful layout model that makes centering elements easy, both vertically and horizontally. css Copy code .container { display: flex; justify-content: center; /* Centers horizontally */ align-items: center; /* Centers vertically */ height: 100vh; /* Optional: set the height of the container */ } .centered-div { width: 200px; /* Width of the centered div */ height: 100px; /* Height of the centered div */ background-color: lightblue; /* Example background */ } HTML: 2. Using Grid Layout CSS Grid is another modern layout method that allows easy centering of elements. css Copy code .container { display: grid; place-items: center; /* Centers both horizontally and vertically */ height: 100vh; /* Optional: set the height of the container */ } .centered-div { width: 200px; /* Width of the centered div */ height: 100px; /* Height of the centered div */ background-color: lightblue; /* Example background */ } HTML: 3. Using Positioning (Absolute) You can use absolute positioning to center the div. This method requires setting the position of the container to relative and the centered div to absolute. css Copy code .container { position: relative; height: 100vh; /* Optional: set the height of the container */ } .centered-div { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); /* Offset the div by 50% of its width and height */ width: 200px; /* Width of the centered div */ height: 100px; /* Height of the centered div */ background-color: lightblue; /* Example background */ } HTML: 4. Using Margin (Auto) If the div has a fixed width, you can use the margin property to center it horizontally. To center it vertically, you can combine it with a known height. css Copy code .container { display: block; height: 100vh; /* Optional: set the height of the container */ } .centered-div { width: 200px; /* Width of the centered div */ height: 100px; /* Height of the centered div */ background-color: lightblue; /* Example background */ margin: auto; /* Horizontally center */ position: relative; top: 50%; /* Vertically center */ transform: translateY(-50%); /* Adjust for exact vertical centering */ } HTML: Summary of Methods: Flexbox: Use display: flex with justify-content: center and align-items: center. Most modern and easiest method. Grid Layout: Use display: grid with place-items: center. Simple and powerful for layout control. Positioning (Absolute): Use position: absolute with top, left, and transform properties. Useful for fixed-size elements. Margin Auto: Use margin: auto for horizontal centering and position: relative with top and transform for vertical centering. Requires fixed sizes. For modern web development, Flexbox and Grid Layout are generally preferred due to their simplicity and flexibility. Document
